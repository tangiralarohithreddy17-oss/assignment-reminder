# Assignly (Assignment Reminder)

A full-stack Assignment Reminder web application built using React, Node.js, Express, and PostgreSQL.

## Features

* User Login & Registration
* Add Assignments
* Due Date & Time Tracking
* Due Today / Upcoming / Overdue Sections
* Assignment Countdown
* Reminder Notifications
* Mark Assignments as Completed
* Group Chat
* Group Members
* Dashboard Analytics

## Tech Stack

Frontend:

* React.js
* React Router
* Axios

Backend:

* Node.js
* Express.js

Database:

* PostgreSQL
